# Cloud Fundamentals

Foundational cloud knowledge (AWS/GCP/Azure) to prepare for infrastructure, automation, and scalable deployments.

---

## 1. Purpose
To learn practical cloud skills that support modern application development and DevOps workflows.

---

## 2. Topics Covered
- Cloud compute (EC2, VM)
- Storage (S3, buckets)
- Networking basics
- IAM & permissions
- Serverless functions
- Containers & Docker basics
- Monitoring & logging
- Cost fundamentals

---

## 3. Practical Exercises
- Deploy a static website
- Upload/read/write files in cloud storage
- Create a simple serverless function
- Test basic IAM roles

---

## 4. Copilot Instructions
Copilot should:
- Provide secure IAM recommendations
- Avoid hard-coded credentials
- Suggest cloud architecture diagrams when asked
- Prioritize least-privilege patterns
