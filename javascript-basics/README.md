# JavaScript Basics

A structured introduction to JavaScript for web development, scripting, and future frontend work.

---

## 1. Purpose
To establish a strong foundation in modern JavaScript (ES6+) and prepare for browser scripting or Node.js prototyping.

---

## 2. Learning Outcomes
- Understand JS variables, types, and operators
- Write functions, loops, and conditionals
- Manipulate arrays and objects
- Understand async programming basics (`promise`, `async/await`)
- Use browser DOM APIs
- Run JS scripts via Node.js

---

## 3. Core Topics
- JS syntax fundamentals
- ES6+ features
- Functions & scope
- Arrays, objects, maps, sets
- Basic DOM manipulation
- Events
- Fetch API
- Async patterns

---

## 4. Starter Exercises
- Simple DOM manipulation script
- Fetch data from an API
- Create a basic counter app
- Write a script that sorts or filters lists

---

## 5. Copilot Instructions
Inside this folder, Copilot should:
- Suggest modern ES6 syntax
- Avoid legacy patterns (`var`, callbacks)
- Provide browser-safe examples
- Demonstrate async/await patterns
