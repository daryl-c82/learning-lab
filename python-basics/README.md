# Python Basics

A foundational track to build core programming skills with Python. This section supports your transition into software development and prepares you for AI engineering, automation, and backend prototyping.

---

## 1. Purpose
To learn Python fundamentals through hands-on exercises that build fluency, confidence, and practical problem-solving skills.

---

## 2. Learning Outcomes
By the end of this track, you will be able to:
- Write clean, readable Python scripts
- Use variables, functions, loops, and conditionals
- Work with lists, dictionaries, sets, and tuples
- Understand Python modules and imports
- Handle files and simple data processing
- Apply debugging techniques
- Build small automation scripts

---

## 3. Core Topics
- Intro to Python
- Control flow
- Functions and modular code
- Data structures
- File I/O
- Error handling
- Working with libraries (`os`, `json`, `requests`)
- Virtual environments & tooling

---

## 4. Starter Exercises
- Print and variables
- Build a simple calculator
- Create a to-do list CLI app
- Read and process a `.txt` file
- Make a basic API request
